Decorative Background Images Demo

Usage of CSS Background Image with No Semantic Meaning:

Decorative images should be applied as CSS background images when they do not convey content or provide functionality. This ensures that assistive technologies ignore them. 

              
                <div class="decorative-image"></div>
                <style>
                  .decorative-image {
                    width: 200px;
                    height: 200px;
                    background-image: url('decorative-pattern.jpg');
                    background-size: cover;
                  }
                </style>

Usage of Hidden Decorative Background Image with ARIA Hidden:

When the decorative background is part of an element that could be interpreted as meaningful, use aria-hidden="true" to prevent it from being announced by screen readers.

              
                <div class="decorative-section" aria-hidden="true"></div>
                <style>
                .decorative-section {
                  width: 100%;
                  height: 300px;
                  background-image: url('decorative-banner.jpg');
                  background-repeat: no-repeat;
                  background-position: center;
                }
                <style>

Usage of background image with role="presentation":

If a background image is purely decorative, ensure it is not announced by assistive technologies. Example: 

              
              <div role="presentation" 
              style="width: 300px; height: 150px; 
              background-image: url('decorative-pattern.jpg'); 
              background-size: cover;"></div>
