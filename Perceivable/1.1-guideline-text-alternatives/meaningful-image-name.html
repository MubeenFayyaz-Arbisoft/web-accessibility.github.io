1.1 Guideline / Meaningful Image Name

<alt> attribute Usage in <img> tag
             {    
             <img alt="arbisoft brand logo" width="125" height="30" style="color:transparent" src="/images/logo.svg">
             }

Usage in HTML:

(<img>) tag and (<alt>) attribute to give text to meaningful description. (<figure>) tag and (<figcaption>) also help AT/ screen reader identify images, giving an (<alt>) in it is also valid as (<figcaption>) will help read text within it.
